# ATIVIDADE PRÁTICA ORIENTADA – UNIDADE 1  
## LINGUAGEM PHP PROCEDURAL E WEB

---

## 1. INTRODUÇÃO

Este repositório contém a resolução das atividades práticas da Unidade 1 da disciplina de Programação Web, com foco na linguagem PHP procedural e sua integração com formulários HTML, sessões e cookies.

As atividades foram desenvolvidas com o objetivo de consolidar conceitos fundamentais da linguagem, permitindo a construção de aplicações web simples e funcionais.

---

## 2. OBJETIVO

### 2.1 Objetivo Geral

Aplicar os conceitos fundamentais de PHP procedural no desenvolvimento de aplicações web.

### 2.2 Objetivos Específicos

- Utilizar estruturas condicionais e de repetição;
- Trabalhar com formulários HTML e método POST;
- Implementar controle de sessão;
- Utilizar cookies para persistência de dados;
- Desenvolver um sistema simples integrando os conceitos estudados.

---

## 3. ESTRUTURA DO PROJETO

O projeto está organizado da seguinte forma:

unidade-1/

├── atividade-1-classificacao/  
├── atividade-2-formulario/  
├── atividade-3-login/  
├── atividade-4-cookies/  
└── atividade-5-desafio-integrador/  

Cada pasta contém os arquivos necessários para execução de cada atividade proposta.

---

## 4. DESCRIÇÃO DAS ATIVIDADES

### 4.1 Atividade 1 – Classificação Acadêmica

Sistema que recebe uma nota e informa a situação do aluno (Aprovado, Recuperação ou Reprovado), além de exibir uma sequência de valores utilizando estrutura de repetição.

---

### 4.2 Atividade 2 – Formulário

Formulário que coleta dados de um aluno, valida os campos e exibe as informações preenchidas.

---

### 4.3 Atividade 3 – Sistema de Login

Sistema de autenticação com uso de sessões, controle de acesso e funcionalidade de logout.

---

### 4.4 Atividade 4 – Cookies

Sistema que armazena o nome do usuário em cookies por 7 dias, exibindo mensagem personalizada em acessos futuros.

---

### 4.5 Atividade 5 – Desafio Integrador

Sistema completo contendo:

- Login com sessão;
- Cadastro de usuários;
- Armazenamento temporário em array;
- Exibição de registros;
- Validação de dados por função.

---

## 5. EXECUÇÃO

Para executar os projetos, utilize o servidor embutido do PHP:

php -S localhost:8000 -t nome-da-pasta

Exemplo:

php -S localhost:8000 -t atividade-2-formulario

Após iniciar o servidor, acesse no navegador:

http://localhost:8000

---

As atividades permitiram o desenvolvimento das competências iniciais em PHP, abrangendo desde conceitos básicos até a criação de sistemas simples com interação do usuário.
