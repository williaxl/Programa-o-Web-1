# ATIVIDADE PRÁTICA – LINGUAGEM PHP

## 1. INTRODUÇÃO

Este repositório foi desenvolvido com o objetivo de apresentar a resolução de exercícios práticos da disciplina de Programação Web, com foco na linguagem PHP.

As atividades propostas têm como finalidade introduzir conceitos fundamentais da linguagem, permitindo ao estudante desenvolver habilidades iniciais em lógica de programação e manipulação de dados em ambiente web.

---

## 2. OBJETIVOS

### 2.1 Objetivo Geral

Desenvolver a compreensão dos conceitos básicos da linguagem PHP por meio da resolução de exercícios práticos.

### 2.2 Objetivos Específicos

- Compreender os tipos de dados em PHP;
- Utilizar operadores aritméticos, lógicos e relacionais;
- Trabalhar com arrays indexados, associativos e multidimensionais;
- Manipular strings utilizando funções nativas da linguagem;
- Entender conceitos básicos de escopo de variáveis;
- Introduzir noções de programação orientada a objetos;
- Desenvolver um formulário simples utilizando PHP.

---

## 3. METODOLOGIA

As atividades foram realizadas individualmente, utilizando o ambiente Termux para execução dos scripts em PHP. Cada questão foi desenvolvida em um arquivo separado, facilitando a organização e a compreensão do conteúdo.

---

## 4. ESTRUTURA DO PROJETO

O projeto está organizado da seguinte forma:

- questao1.php
- questao2.php
- questao3.php
- ...
- questao21.php

Cada arquivo contém a resolução de uma questão específica proposta na atividade.

---

## 5. EXECUÇÃO

Para executar os arquivos PHP, utilize o seguinte comando no terminal:

php nome_do_arquivo.php

Exemplo:

php questao1.php

---

## 6. OBSERVAÇÃO

A questão 21 envolve a utilização de formulário HTML com PHP. Para sua execução, é necessário utilizar um servidor local, conforme o comando abaixo:

php -S localhost:8000

Após iniciar o servidor, acesse no navegador:

http://localhost:8000/questao21.php

---

## 7. CONSIDERAÇÕES FINAIS

A realização desta atividade contribuiu para a compreensão inicial da linguagem PHP, permitindo o desenvolvimento de habilidades básicas necessárias para a construção de aplicações web.
